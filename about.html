About dawg.city

dawg.city is a free, public tool to help anyone instantly detect whether a YouTube video is AI-generated or authentic. In a world increasingly flooded with synthetic media, we believe everyone deserves access to content verification tools.

Our Mission

The internet is changing fast. AI-generated videos, deepfakes, and synthetic media are becoming indistinguishable from real content at an alarming rate. dawg.city was built with one goal: make AI content detection fast, free, and accessible to everyone — no technical knowledge required.

We believe informed audiences are the best defense against AI misinformation. By putting a powerful detection tool directly in anyone's browser, we help people make better decisions about the content they watch and share.

How the Detection Works

dawg.city analyzes YouTube video thumbnails using Sightengine's AI-Generated content detection API. When you paste a YouTube URL, our backend extracts the high-resolution thumbnail image and sends it to Sightengine's GenAI model, which returns a probability score indicating how likely the content is to be AI-generated.

Based on this score, we return a binary verdict: Fake (likely AI-generated or synthetic) or Real (likely authentic, human-created content), along with a confidence percentage.

Note: dawg.city analyzes thumbnails as a proxy for video content. While thumbnails are a strong signal for AI-generated videos, they are not a substitute for full video analysis.

Who Built This

dawg.city is an independent project built by a solo developer passionate about media authenticity and AI transparency. It is not affiliated with YouTube, Google, or any major technology company.

The tool is built on a Rust backend deployed on Vercel, using the Sightengine API for AI detection. The frontend is pure HTML, CSS, and JavaScript — no frameworks, fast loading.
